Poissonization-based collision threshold derivation for random walks on lattices

This paper studies the threshold behavior of collisions in a system of independent random walks on $\mathbb{Z}^3$, using Poissonization and generating function asymptotics. We derive a sharp threshold for the expected number of pairwise collisions and establish asymptotic expressions via the Laplace method.
